# Pilha Java

A brief and concise description of what this Java application does and its main purpose.

## Project Overview

This project is a Java-based application developed using the IntelliJ IDEA environment. It follows a standard project structure for standalone Java applications, focusing on clean code and modularity.

## Prerequisites

Before you begin, ensure you have the following installed:
* Java Development Kit (JDK) 11 or higher (JDK 17+ recommended).
* IntelliJ IDEA (Community or Ultimate edition).
* Git (optional, for version control).

## Project Structure

The project follows a standard directory layout:

* **src**: Contains all Java source files (.java).
* **lib**: Contains external dependencies and third-party libraries (.jar files).
* **out**: The default directory where compiled bytecode (.class files) is generated by IntelliJ IDEA.
